| INSTALL BOT |
|:-----------------------|
- <p align="left">Clone Source:
```sh
git clone https://github.com/EleGanceTeam/EleGanceTM
```
- <p align="left">First install:
```sh
cd EleGanceTM
chmod 777 Elegance
./Elegance.sh
```
- <p align="left">install bot:
```sh
chmod 777 launch.sh
./launch.sh
```
| ANTI CRASH |
|:-----------------------|
```sh
tmux new-session -s script "bash steady.sh -t"
```

`./start.sh -q`

| FREE SERVER: |
